后代选择器(重要)

  1. 元素1 元素2{

    可以是任意的基础选择器

    }

  2. ol li{

    只选择ol里面的li

    }

  3. ol li a{

    }

  4. .classname li a{

    根据类名选择

    }

子选择器(重要)

    只选择儿子,不能选孙子

  1. .nav > a{

    大于号分割 >

    }

    }
  2. div,
    p{

    同时更改div和p标签的效果

    }

    p,

    }

伪类选择器

链接伪类选择器

添加特殊效果

    一定要安装顺序写

  1. a:link 选择所有未被访问的链接
  2. a:visited 选择所有已被访问的链接
  3. a:hover 选择鼠标指针位于其上的链接
  4. a:active 选择活动的链接(鼠标按下未弹起的链接)
